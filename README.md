# Fear_Greed_Trading_Analysis

## Objective

The goal of this task is to analyze the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data.

## Dataset

- Historical Trading Data
- Bitcoin Fear & Greed Index

## Steps Performed

- Loaded both datasets
- Converted timestamps to datetime format
- Extracted dates from historical data
- Merged the datasets using the date column
- Checked for missing and duplicate values
- Performed exploratory data analysis (EDA)
- Visualized trading performance across different market sentiment categories

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insight

The analysis shows that trader performance varies across different market sentiment categories. Average Closed PnL was highest during **Extreme Greed** and lowest during **Extreme Fear**.
